# Crypto Prediction Hub - Product Requirements Document

## User Request
Create a cryptocurrency prediction website that provides Long/Short trading signals with real logic and real prices. The platform should analyze news, trends, and market data to generate accurate predictions with detailed reasoning.

## Core Requirements

### Functional Requirements
1. Display real-time cryptocurrency prices from live API
2. Generate Long/Short predictions based on technical analysis
3. Provide detailed commentary explaining each prediction
4. Show confidence scores and sentiment analysis
5. Track market trends and price movements
6. Display multiple cryptocurrencies with predictions

### Technical Requirements
1. Use CoinGecko API for real-time price data
2. Implement prediction engine with technical indicators:
   - Price trends (7d, 24h)
   - Volume analysis
   - Market cap changes
   - Volatility indicators
3. Calculate sentiment scores based on:
   - Price momentum
   - Volume patterns
   - Market cap trends
4. Type-safe API integration
5. Responsive design for all devices

### Design Requirements
1. Professional trading platform aesthetic
2. Clear visual distinction between Long (green) and Short (red) signals
3. Real-time data updates
4. Easy-to-read prediction cards
5. Market overview dashboard
6. Sentiment indicators with color coding

## Success Criteria
- Real prices updating from live API
- Accurate technical analysis with valid logic
- Clear Long/Short recommendations
- Professional, polished UI
- Fast performance and smooth interactions
